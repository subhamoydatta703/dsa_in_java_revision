1. Method Scope

Definition:
Variables declared inside a method can be accessed only within that method.
They are created when the method is called and destroyed when it ends.

‚úÖ Example:

public class MethodScopeExample {
    public static void main(String[] args) {
        greet();  // calling the method
        // System.out.println(message); ‚ùå Error: message not visible here
    }

    static void greet() {
        String message = "Hello from greet()!"; // variable declared inside method
        System.out.println(message); // ‚úÖ Works fine
    }
}

The variable message is declared inside the method greet().

It‚Äôs accessible only inside greet().

Once the method ends, message is destroyed.

 2. Block Scope

Definition:
Variables declared inside any block { ... } ‚Äî like if, for, or while ‚Äî are accessible only inside that block.

‚úÖ Example:

public class BlockScopeExample {
    public static void main(String[] args) {
        int x = 10;

        if (x > 5) {
            int y = 20; // variable inside if-block
            System.out.println("x + y = " + (x + y)); // ‚úÖ Works fine
        }

        // System.out.println(y); ‚ùå Error: y is not visible outside the block
    }
}


üü© Explanation:

Variable y is declared inside the if block.

It cannot be used outside the block {} where it‚Äôs declared.

The lifetime of y ends when the block ends.
